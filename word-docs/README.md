# Panduan Penggunaan Dokumen Word

Folder ini berisi dokumen dalam format Markdown yang dirancang agar mudah dipindahkan ke aplikasi Microsoft Word.

## Langkah-langkah konversi ke Word:

### 1. Menggunakan file "Master Prompt"
Jika anda ingin prompt lengkap yang mencakup alur dan kamus data sekaligus, gunakan file `PROMPT_SISTEM_SAKTI.md`.

### 2. Mengubah Kamus Data ke Word
*   Buka file `Kamus_Data.md`.
*   Salin isinya dan tempel ke Microsoft Word.

### 3. Membuat Gambar Flowchart
*   Buka file `Flowchart.md`.
*   Salin kode Mermaid dan tempel ke [Mermaid Live Editor](https://mermaid.live/).
*   Unduh gambarnya untuk dimasukkan ke Word.
